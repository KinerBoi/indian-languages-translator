# indian-languages-translator

Essentially a Translating Website that contains audio, text and picture recognition

Sign in:
- Photo
  - Put in photo
  - Choose language
  - Have output
  - Store history
- Text
  - Put in text like Chatgpt
  - Choose language
  - Have output
  - Store history
- Audio
   - Put in audio/speak
   - Choose language
   - Have output
   - Store history
